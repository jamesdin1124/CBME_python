## 1. 系統概述

本系統為兒科住院醫師能力導向醫學教育（CBME）的線上評核平台，提供三大類評核表單：

| 表單類型 | 說明 | 填寫者 |
|---------|------|--------|
| **操作技術** | 16 項臨床操作技能（插管、超音波等） | 主治醫師 |
| **會議報告** | 五維度評分（內容、辯證、表達等） | 主治醫師 |
| **EPA 評估** | 5 項可信賴專業活動 | 主治醫師 |

### EPA 項目一覽

| EPA 項目 | 說明 |
|---------|------|
| 門診表現(OPD) | 門診問診、衛教、處方能力 |
| 一般病人照護（WARD） | 病房日常查房、醫囑、照護計畫 |
| 緊急處置（ED, DR） | 急診與產房緊急處置能力 |
| 重症照護（PICU, NICU） | 加護病房照護與監測能力 |
| 病歷書寫 | SOAP 格式、診斷推理記錄品質 |

## 2. 帳號註冊與登入

### 2.1 申請帳號

1. 開啟系統首頁，點選上方 **「申請帳號」** 分頁

<!-- 截圖位置：登入/申請帳號頁面 -->
![申請帳號頁面](screenshots/registration_page.png)

2. 填寫以下必填欄位：
   - **真實姓名**：請填寫全名
   - **帳號**：英文或數字組合
   - **密碼**：請輸入兩次確認
   - **Email**：公務信箱
   - **科別**：選擇「小兒部」
   - **身份**：
     - 住院醫師選擇 **「住院醫師」**，並填寫級職（R1/R2/R3）與指導醫師
     - 主治醫師選擇 **「主治醫師」**

3. 點選 **「提交申請」**，等待管理員審核通過後即可登入

### 2.2 登入系統

1. 開啟系統首頁，在 **「登入」** 分頁中輸入帳號與密碼
2. 點選 **「登入」** 按鈕
3. 登入成功後將自動跳轉至主頁面

## 3. 填寫評核表單（主治醫師）

登入後，在系統導覽中進入 **「線上評核表單」** 區域，可看到三個分頁：

### 3.1 操作技術評核

用於記錄住院醫師執行臨床操作的表現。

**填寫步驟：**

1. 點選 **「操作技術」** 分頁
2. 填寫共用欄位：
   - **評核日期**：預設為今天，可修改
   - **受評核人員**：從下拉選單選擇住院醫師
   - **評核時級職**：系統自動帶入，可手動修改
3. 選擇 **評核技術項目**（如：插氣管內管、腰椎穿刺等），括號中會顯示最低要求次數
4. 填寫 **病歷號**（必填）
5. 填寫 **鎮靜藥物**（選填）
6. 選擇 **可信賴程度**（9 級量表，詳見下表）
7. 填寫 **操作技術教師回饋**
8. 點選 **「提交技術評核」**

<!-- 截圖位置：操作技術表單 -->
![操作技術表單](screenshots/technical_skill_form.png)

**16 項操作技術與最低要求次數：**

| 技能項目 | 最低次數 | 技能項目 | 最低次數 |
|---------|---------|---------|---------|
| 插氣管內管 | 3 | 腦部超音波 | 5 |
| 插臍(動靜脈)導管 | 1 | 心臟超音波 | 5 |
| 腰椎穿刺 | 3 | 腹部超音波 | 5 |
| 插中心靜脈導管(CVC) | 3 | 腎臟超音波 | 5 |
| 肋膜液或是腹水抽取 | 1 | APLS | 3 |
| 插胸管 | 2 | NRP | 5 |
| 放置動脈導管 | 2 | CVVH照護 | 1 |
| 經皮式中央靜脈導管(PICC) | 3 | ECMO照護 | 1 |

### 3.2 會議報告評核

用於評核住院醫師在各類會議中的報告表現。

**填寫步驟：**

1. 點選 **「會議報告」** 分頁
2. 填寫共用欄位（日期、受評核人員、級職）
3. 從下拉選單選擇 **會議名稱**：
   - Staff Round
   - Journal Meeting
   - 晨會指導
   - EBM指導
   - 多專科會議
   - MM
   - 其他
4. 填寫 **報告主題**（如：新生兒呼吸窘迫症候群處置）
5. 針對五個維度進行 **1-5 分評分**（拖動滑桿）：
   - 內容是否充分
   - 辯證資料的能力
   - 口條、呈現方式是否清晰
   - 是否具開創、建設性的想法
   - 回答提問是否具邏輯、有條有理
6. 填寫 **會議報告教師回饋**
7. 點選 **「提交會議報告評核」**

<!-- 截圖位置：會議報告表單 -->
![會議報告表單](screenshots/meeting_report_form.png)

**評分說明：**

| 分數 | 等級 | 說明 |
|------|------|------|
| 5 | 卓越 | 表現超越期待，可作為範例 |
| 4 | 充分 | 表現符合或略超期待 |
| 3 | 尚可 | 基本符合期待，仍有進步空間 |
| 2 | 稍差 | 未完全達到期待 |
| 1 | 不符合期待 | 需明顯改善 |

### 3.3 EPA 信賴等級評估

用於評估住院醫師在各 EPA 項目中的可信賴程度。

**填寫步驟：**

1. 點選 **「EPA 評估」** 分頁
2. 填寫共用欄位（日期、受評核人員、級職）
3. 選擇 **EPA 項目**：
   - 門診表現(OPD)
   - 一般病人照護（WARD）
   - 緊急處置（ED, DR）
   - 重症照護（PICU, NICU）
   - 病歷書寫
4. 選擇 **可信賴程度**（以下 9 級量表，以 radio button 選擇）
5. 填寫 **EPA 質性回饋**
6. 點選 **「提交 EPA 評估」**

<!-- 截圖位置：EPA 評估表單 -->
![EPA評估表單](screenshots/epa_form.png)

### 可信賴程度量表（操作技術 / EPA 共用）

| 等級 | 描述 |
|------|------|
| 1 | 允許住院醫師在旁觀察 |
| 2 | 教師在旁逐步共同操作 |
| 3 | 教師在旁必要時協助 |
| 4 | 教師可立即到場協助，事後逐項確認 |
| 5 | 教師可立即到場協助，事後重點確認 |
| 6 | 教師可稍後到場協助，必要時事後確認 |
| 7 | 教師on call提供監督 |
| 8 | 教師不需on call，事後提供回饋及監督 |
| 9 | 學員可對其他資淺的學員進行監督與教學 |

## 4. 住院醫師自填表單

住院醫師登入後可在 **「住院醫師自填表單」** 區域填寫：

### 4.1 研究進度登記

記錄個人研究計畫的進度，包含：
- 研究名稱、文獻類型、指導老師
- 目前進度（構思中/撰寫中/投稿中/接受）
- 目標期刊、進度說明

### 4.2 學習反思記錄

以 Gibbs 反思循環架構記錄臨床學習經驗：
1. 情境描述
2. 想法與感受
3. 評估與分析
4. 行動計畫
5. 學習成果

可關聯相關 EPA 項目和技能項目。

## 6. CCC 會議流程

CCC（Clinical Competency Committee，臨床能力委員會）為定期召開的住院醫師訓練評核會議。

### 6.1 會前準備

1. **管理員/計畫主持人**登入系統，進入 CCC 總覽頁面
2. 檢視所有住院醫師的訓練進度狀態（綠/黃/紅燈分類）
3. 準備需要討論的住院醫師名單（重點關注紅燈與黃燈住院醫師）

### 6.2 會議進行

1. **逐一檢視**每位住院醫師的個別分析報告：
   - 操作技能完成度（是否達到最低要求次數）
   - EPA 可信賴程度趨勢（是否有進步）
   - 會議報告五維度平均分數
   - 教師質性回饋彙整
2. **委員討論**住院醫師的整體表現
3. **記錄決議**：針對每位住院醫師做出判定（詳見第 7 章）

### 6.3 會後追蹤

1. 將 CCC 會議決議通知住院醫師與指導醫師
2. 針對需輔導的住院醫師，制定具體改善計畫
3. 下次 CCC 會議追蹤改善進度

## 7. 評定住院醫師是否合資

### 7.1 評定標準

住院醫師是否合資（通過評核）依據以下三大面向綜合判定：

#### (1) 操作技術

- 所有 16 項操作技術須達到 **最低要求次數**
- 近期評核的可信賴程度應有進步趨勢

#### (2) EPA 可信賴程度

各 EPA 項目須達到對應訓練年級的預期可信賴等級：

| 訓練年級 | 預期可信賴等級 |
|---------|-------------|
| R1 | 教師在旁必要時協助（等級 3）以上 |
| R2 | 教師可立即到場協助，事後重點確認（等級 5）以上 |
| R3 | 教師on call提供監督（等級 7）以上 |

#### (3) 會議報告

- 五維度平均分數應達 **3.0 分以上**
- 無單一維度持續低於 2 分

### 7.2 判定結果分類

| 判定結果 | 說明 |
|---------|------|
| **合資（通過）** | 所有面向均達到期待，可進入下一訓練階段 |
| **有條件通過** | 大部分面向達標，但有特定項目需限期改善 |
| **暫不通過** | 多項指標未達標，需制定輔導計畫並延後晉升 |

### 7.3 合資判定流程

```
收集評核數據 → CCC 會議討論 → 綜合判定 → 通知住院醫師
                                ↓
                         合資 → 進入下一階段
                         有條件通過 → 限期改善 → 追蹤覆評
                         暫不通過 → 輔導計畫 → 下次 CCC 覆評
```

## 9. 常見問題

### Q: 忘記密碼怎麼辦？
A: 請聯繫科別管理員協助重設密碼。

### Q: 提交表單後發現填錯資料，可以修改嗎？
A: 目前系統不支援修改已提交的評核紀錄，請聯繫管理員處理。如需刪除錯誤紀錄，請提供評核日期、受評核人員等資訊。

### Q: 可信賴程度選哪一個等級？
A: 請依據實際觀察到的住院醫師獨立執行程度選擇。重點是「若再次遇到類似情境，該住院醫師需要多少程度的監督」。

### Q: 會議報告的「其他」會議名稱如何使用？
A: 當會議類型不在預設選項中時，選擇「其他」，並在報告主題欄位中說明實際會議名稱與內容。

### Q: 住院醫師可以看到自己的評核分數嗎？
A: 住院醫師可以看到自己的學習歷程與進度，但具體的教師質性回饋需由指導醫師在面談時提供。
