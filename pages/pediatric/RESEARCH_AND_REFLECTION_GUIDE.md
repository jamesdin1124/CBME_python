# 兒科住院醫師研究進度與學習反思系統使用指南

## 📋 系統概述

本系統新增了兩大功能模組：
1. **研究進度追蹤**：記錄住院醫師的研究計畫進度（文獻回顧、個案報告、原著論文等）
2. **學習反思記錄**：基於 Gibbs 反思循環架構，記錄臨床學習經驗與反思

---

## 🚀 快速開始

### 1. 資料庫設定（首次使用）

在 Supabase SQL Editor 中執行以下 SQL 腳本：

```bash
# 在 Supabase 中執行
sql/pediatric_research_schema.sql
```

這會建立兩個資料表：
- `pediatric_research_progress`：研究進度記錄
- `pediatric_learning_reflections`：學習反思記錄

## 👥 使用者角色與權限

### 住院醫師 (Resident)

## 📝 功能詳解

### 功能 1：研究進度追蹤

#### 填寫位置
住院醫師登入後 → **「我的表單」分頁** → **「📚 研究進度」**

#### 表單欄位
- **研究名稱** *（必填）：研究計畫名稱
- **文獻類型** *（必填）：個案報告 / 原著論文 / 系統性回顧 / 文獻回顧 / 其他
- **指導老師**：從教師名單中選擇
- **目前進度** *（必填）：構思中 / 撰寫中 / 投稿中 / 接受

**詳細資訊（選填）：**
- 研究主題/領域
- 目標期刊
- 投稿日期、接受日期
- 進度說明
- 遭遇困難
- 下一步計畫

#### 顯示位置
1. **CCC 總覽頁面**
   - 摘要卡片底部顯示最新研究進度
   - 頁面底部「研究進度總覽」區塊顯示所有住院醫師的研究統計

2. **個別深入分析頁面**
   - Section F：研究進度完整列表
   - 統計卡片（構思中、撰寫中、投稿中、接受）
   - 詳細進度說明（可展開）

---

### 功能 2：學習反思記錄

#### 填寫位置
住院醫師登入後 → **「我的表單」分頁** → **「💭 學習反思」**

#### 表單欄位（Gibbs 反思循環架構）
- **反思日期** *（必填）
- **反思類型** *（必填）：臨床反思 / 學習心得 / 個案討論 / 技能學習 / 其他
- **反思標題** *（必填）
- **設為私人記錄**：勾選後僅自己可見

**反思內容（Gibbs 五階段）：**
1. **情境描述**：發生了什麼事？
2. **想法與感受**：當時的想法與感受是什麼？
3. **評估與分析**：什麼做得好？什麼可以改進？
4. **行動計畫**：下次遇到類似情況會怎麼做？
5. **學習成果**：從這次經驗中學到了什麼？

**關聯資訊（選填）：**
- 相關 EPA 項目
- 相關技能項目
- 督導教師
- 標籤（溝通技巧、臨床決策、團隊合作等）

#### 顯示位置
1. **個別深入分析頁面**
   - Section G：學習反思記錄
   - 類型統計
   - 最新 10 筆反思清單
   - 詳細反思內容（可展開，顯示完整 Gibbs 五階段）

## 🎯 使用情境範例

### 情境 1：研究進度登記
**R2 住院醫師 李小明** 正在進行個案報告：

1. 登入系統 → 「我的表單」→ 「研究進度」
2. 填寫：
   - 研究名稱：新生兒敗血症之臨床分析
   - 文獻類型：個案報告
   - 指導老師：王主任
   - 目前進度：撰寫中
   - 進度說明：已完成文獻回顧，正在撰寫病例報告部分
   - 遭遇困難：統計分析部分需要協助
3. 提交後，在 CCC 總覽與個人分析頁面即可看到此研究進度

### 情境 2：臨床反思記錄
**R1 住院醫師 陳小華** 第一次獨立執行腰椎穿刺：

1. 登入系統 → 「我的表單」→ 「學習反思」
2. 填寫：
   - 反思標題：第一次獨立執行腰椎穿刺的經驗
   - 反思類型：技能學習
   - 相關技能：腰椎穿刺
   - 督導教師：張醫師
3. 依 Gibbs 架構填寫五階段內容
4. 選擇標籤：臨床決策、壓力調適
5. 提交後，在個人分析頁面可看到完整反思記錄

## 📊 資料管理

### 資料更新
- 研究進度與學習反思皆儲存於 Supabase
- 資料即時同步，填寫後立即在各頁面顯示
- 支援軟刪除（`is_deleted = TRUE`），資料不會真正刪除

### 隱私設定
- 學習反思可設為「私人記錄」
- 私人記錄目前僅在本人的「我的表單」中顯示
- 未來可擴充為依使用者權限控制顯示範圍

## 🔧 技術架構

### 資料庫結構
```sql
-- 研究進度表
CREATE TABLE pediatric_research_progress (
    id, created_at, updated_at,
    resident_name, resident_level,
    research_title, research_type, supervisor_name,
    current_status, research_topic, target_journal,
    submission_date, acceptance_date, publication_date, doi,
    progress_notes, challenges, next_steps,
    is_deleted, submitted_by
);

-- 學習反思表
CREATE TABLE pediatric_learning_reflections (
    id, created_at, updated_at,
    resident_name, resident_level,
    reflection_date, reflection_title, reflection_type,
    situation_description, thoughts_and_feelings,
    evaluation, action_plan, learning_outcomes,
    related_epa, related_skill, supervising_teacher,
    tags[], is_private, is_deleted, submitted_by
);
```

### 系統模組
- **supabase_connection.py**：新增研究進度與學習反思的 CRUD 方法
- **pediatric_resident_forms.py**：住院醫師自填表單介面
- **pediatric_analysis.py**：在 CCC 總覽與個人分析頁面顯示資料

### 頁面層級
```
兒科住院醫師評核系統
├─ CCC 總覽（所有住院醫師）
│  ├─ 摘要卡片（顯示最新研究）
│  └─ 研究進度總覽（統計與清單）
├─ 個別深入分析（單一住院醫師）
│  ├─ Section F：研究進度
│  └─ Section G：學習反思記錄
└─ 我的表單（住院醫師限定）
   ├─ 研究進度填寫
   └─ 學習反思填寫
```

## ❓ 常見問題

### Q1：如何修改已提交的研究進度？
目前系統支援新增，暫不支援線上編輯。如需修改，請聯繫管理員或在資料庫中直接更新。

### Q2：私人反思記錄會被教師看到嗎？
目前私人記錄在個人分析頁面不顯示，僅住院醫師本人在「我的表單」中可見。

### Q3：研究完成發表後如何更新？
可新增一筆相同研究名稱的記錄，將進度更新為「接受」，並填寫發表日期、DOI 等資訊。

### Q4：如何查看歷史反思記錄？
在「個別深入分析」頁面，選擇住院醫師後，捲動到「學習反思記錄」區塊，展開「詳細反思內容」即可查看。
